# AprovaIA — Landing Page

Landing page premium do AprovaIA. React 19 + Vite + TypeScript + Tailwind.

## Rodar local
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Deploy na Vercel
1. Suba esta pasta para um repositório GitHub novo (separado do app)
2. vercel.com/new → Import → selecione o repositório → Deploy
3. Pronto — o vercel.json já está configurado (npm, sem pnpm-lock)
